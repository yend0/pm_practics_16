# COSMIC Function Points (CFP)

## Тип операции:

- E (Entry) — ввод данных в систему
- X (Exit) — вывод данных системой
- R (Read) — чтение данных
- W (Write) — запись данных

---

## UC1 — Импорт каталога поставщика

| № | Подзадача | Тип операции |
|---|----------|--------------|
| 1 | Загрузка файла / указание URL каталога | E |
| 2 | Чтение данных каталога | R |
| 3 | Валидация данных по справочникам | R |
| 4 | Сохранение каталога в БД | W |
| 5 | Обновление индексов и связей | R + W |
| 6 | Формирование и отображение отчёта | R + X |

**UC1 CFP = 8**
