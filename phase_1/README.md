# Фаза 1. Анализ и выбор проектной концепции

## 1. Цель этапа

На первом этапе проводится предварительная аналитика проектных концепций и отбор наиболее перспективной идеи для дальнейшей реализации.  
Основная задача — сформировать набор возможных альтернатив, определить критерии оценки и собрать экспертные данные, необходимые для применения методов **TOPSIS** и **МАИ** на следующей фазе.

---

## 2. Описание рассматриваемых концепций

**A1. LifeTracker**  
Система для отслеживания и анализа личных привычек и повседневной активности. Включает ведение статистики, напоминания, визуализацию прогресса и рекомендации по улучшению продуктивности.

**A2. KnitWit**  
Платформа для совместного обучения и ведения образовательных проектов. Обеспечивает управление задачами, коммуникацию участников и интеграцию с внешними образовательными ресурсами.

**A3. Shader Generator.Pro**  
Инструмент для автоматизированного создания и визуализации шейдеров. Поддерживает различные графические движки и ориентирован на разработчиков игр и дизайнеров визуальных эффектов.

**A4. Auto Import Catalog**  
Сервис для автоматического анализа, структурирования и сравнения предложений по автозапчастям. Позволяет импортировать каталоги поставщиков и формировать агрегированные предложения для клиентов.

---

## 3. Критерии оценки

Для оценки инвестиционной привлекательности каждой концепции выбраны следующие критерии:

| № | Критерий | Тип | Описание |
|---|-----------|------|-----------|
| **K1** | Потенциал прибыли | benefit | Способность проекта приносить доход: модель монетизации, платёжеспособность аудитории, масштабируемость. |
| **K2** | Затраты на разработку и поддержку | cost | Объём требуемых ресурсов: трудозатраты, сложность реализации, поддержка и инфраструктура. |
| **K3** | Рыночная привлекательность | benefit | Степень востребованности идеи рынком, актуальность решаемой проблемы, наличие конкурентов. |
| **K4** | Сложность реализации | cost | Вероятность возникновения технических трудностей, зависимость от новых или нестабильных технологий. |
| **K5** | Потенциал масштабирования и роста | benefit | Возможность расширения продукта и роста выручки без значительного увеличения затрат. |
